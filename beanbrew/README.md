# ☕ Bean Brew — Sipariş Yönetimi

React + Tailwind CSS ile geliştirilmiş kahve sipariş yönetim uygulaması.

## Özellikler

- ➕ **Ekle** — Yeni kahve ürünü ekle
- 📋 **Listele** — Tüm kahveleri kartlarla görüntüle
- ✏️ **Güncelle** — Mevcut kahveyi düzenle
- 🗑️ **Sil** — Kahveyi listeden kaldır
- 🔍 **Ara & Filtrele** — İsme veya kategoriye göre filtrele

## Kullanılan Teknolojiler

- ReactJS (useState, useEffect)
- Tailwind CSS
